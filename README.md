# 🧐 Monocle

GitHub 공개 저장소를 검색하고, 등록하고, 이슈를 모아 봅니다.

## ⚙️ 설치

> 프로젝트 클론 이후 패키지 설치 진행

```bash
git clone https://github.com/sunaerocket/monocle.git
```

```bash
cd monocle && npm install
```

## 💻 명령어

> 명령어는 설치가 성공하고 난 후에 작동합니다

- 시작: `npm run dev`
- 빌드: `npm run build`
- 코드검사: `npm run check:code`
